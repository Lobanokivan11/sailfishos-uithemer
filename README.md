# UI themer

With UI themer you can customize icons, fonts and pixel density in Sailfish OS. This app is a GUI for [Theme pack support for Sailfish OS](https://github.com/fravaccaro/themepacksupport-sailfishos).

Based on [Icon pack support GUI](https://github.com/RikudouSage/sailfish-iconpacksupport-gui).
Icon design by [Freepik](http://www.freepik.com/free-photo/blue-paint-roller_959191.htm).
